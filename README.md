Painel de Gestão Operacional (RH & Logística)
Descrição:
Este projeto consiste em uma suíte de ferramentas web desenvolvida para automatizar cálculos rotineiros de departamentos de Recursos Humanos e Logística. O sistema resolve o problema de erros manuais em cálculos de folha de pagamento e fretes, oferecendo interfaces precisas para horas extras, reajustes salariais por faixa e custos de transporte fretado.

Funcionalidades
Cálculo de Horas Extras: Processamento de horas úteis e horas de fins de semana com adicional de 50%.

Reajuste Salarial Automático: Aplicação de porcentagens de aumento baseadas em quatro faixas salariais distintas.

Cálculo de Transporte: Estimativa de custo mensal de fretamento baseado na quantidade de colaboradores e dias úteis.

Data e Hora em Tempo Real: Exibição dinâmica do horário do sistema utilizando o objeto Date do JavaScript (Desafio 2).

Navegação Integrada: Dashboard central (index) que conecta todos os módulos em uma única interface.
